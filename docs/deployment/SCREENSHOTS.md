# Скриншоты развертывания

Сохранять скриншоты в папку:

```text

```

## Список скриншотов

1. `01-docker-compose-ps.png` - результат команды `docker compose ps`.
2. `02-api-health.png` - страница `http://localhost:8080/api/test/health`.
3. `03-swagger.png` - страница `http://localhost:8080/swagger`.
4. `04-react-home.png` - страница `http://localhost:3000`.
5. `05-docker-desktop-containers.png` - список контейнеров в Docker Desktop.

Эти скриншоты подтверждают, что проект развернут через Docker Compose и работает локально перед переносом на выделенный сервер.
