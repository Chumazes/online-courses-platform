# QA-чек-лист: OnlineCourses

## Smoke-проверка

- [x] API запускается командой `dotnet run --launch-profile http`.
- [x] Swagger доступен по адресу `http://localhost:5064/swagger`.
- [x] SQLite-база `OnlineCourses.API/online-courses-dev.db` создается автоматически.
- [x] `GET /api/Courses` возвращает список курсов без авторизации.
- [x] `GET /api/Courses/categories` возвращает список категорий без авторизации.
- [x] Пользователь `student@local.dev / 123456` успешно входит в систему.
- [x] Пользователь `teacher@local.dev / 123456` успешно входит в систему.
- [x] Пользователь `admin@local.dev / 123456` успешно входит в систему.
- [x] `GET /api/Auth/me` с токеном возвращает профиль пользователя.
- [x] `GET /api/Auth/me` без токена возвращает `401 Unauthorized`.
- [x] Desktop/Web-клиент открывает каталог курсов.

## Авторизация и регистрация

- [x] Регистрация нового пользователя с уникальным email проходит успешно.
- [x] Повторная регистрация с тем же email возвращает `400 Bad Request`.
- [x] Вход с правильными данными возвращает access token и refresh token.
- [x] Вход с неверным паролем возвращает `401 Unauthorized`.
- [x] Refresh token обновляет пару токенов.
- [x] Logout отзывает текущий refresh token.
- [x] Logout all отзывает все refresh token пользователя.

## Роли и доступы

- [x] `student` не может создавать курс.
- [x] `teacher` может создать курс.
- [x] `teacher` может редактировать только свой курс.
- [x] `teacher` не может редактировать чужой курс.
- [x] `admin` может управлять категориями.
- [x] `admin` может модерировать отзывы.
- [x] Защищенные POST/PUT/DELETE endpoint без токена возвращают `401`.
- [x] Запрос с недостаточной ролью возвращает `403`.

## Курсы

- [x] Список курсов поддерживает пагинацию.
- [x] Фильтр по уровню курса работает.
- [x] Фильтр по категории работает.
- [x] Поиск по названию или описанию работает.
- [x] Сортировка по рейтингу работает.
- [x] `GET /api/Courses/{id}` возвращает существующий курс.
- [x] `GET /api/Courses/{id}` для несуществующего курса возвращает `404`.
- [x] Создание курса с валидным телом возвращает `201 Created`.
- [x] Обновление курса меняет поля курса.
- [x] Удаление курса удаляет курс из списка.

## Разделы и уроки

- [x] `GET /api/courses/{courseId}/sections` возвращает разделы курса.
- [x] Создание раздела доступно преподавателю или администратору.
- [x] Обновление раздела сохраняет новое название и описание.
- [x] Удаление раздела удаляет связанные уроки или корректно обрабатывает ограничение БД.
- [x] `GET /api/sections/{sectionId}/lessons` возвращает уроки раздела.
- [x] Создание урока доступно преподавателю или администратору.
- [x] Обновление урока меняет название, тип, порядок и контент.
- [x] Удаление урока удаляет урок из раздела.

## Запись на курсы

- [x] Авторизованный студент может записаться на опубликованный курс.
- [x] Повторная запись на тот же курс возвращает ошибку.
- [x] `GET /api/Enrollments/my` возвращает курсы текущего пользователя.
- [x] Пользователь может отписаться от курса.
- [x] Преподаватель может посмотреть студентов своего курса.
- [x] Студент не может посмотреть список студентов курса.

## Прогресс

- [x] Прогресс по уроку создается при первом обновлении.
- [x] Повторное обновление прогресса меняет `isCompleted` и `watchTime`.
- [x] Прогресс курса пересчитывается по завершенным урокам.
- [x] Нельзя обновить прогресс по курсу, на который пользователь не записан.
- [x] `GET /api/Progress/my` возвращает прогресс по всем курсам пользователя.

## Отзывы

- [x] Пользователь может оставить отзыв к курсу.
- [x] Повторный отзыв к тому же курсу от того же пользователя запрещен.
- [x] Новый отзыв требует модерации.
- [x] Публичный список отзывов показывает только одобренные отзывы.
- [x] Администратор может открыть список отзывов для модерации.
- [x] Администратор может одобрить отзыв.
- [x] Рейтинг курса пересчитывается после изменения отзывов.

## Файлы

- [x] Авторизованный пользователь может загрузить аватар допустимого расширения.
- [x] Загрузка аватара недопустимого расширения отклоняется.
- [x] Загрузка слишком большого аватара отклоняется.
- [x] Преподаватель может загрузить файл к своему уроку.
- [x] Студент не может загрузить файл к уроку.
- [x] Файл урока можно скачать через `GET /api/files/download`.
- [x] Скачивание несуществующего файла возвращает ошибку.

## Регрессия

- [x] После создания курса он отображается в общем списке.
- [x] После изменения курса сбрасывается кэш списка курсов.
- [x] После удаления курса он не открывается по ID.
- [x] После создания категории она доступна при создании курса.
- [x] После удаления категории курсы не теряют целостность данных.
- [x] После обновления профиля новые данные видны в `GET /api/Auth/me`.
- [x] После перезапуска API SQLite-данные сохраняются.
